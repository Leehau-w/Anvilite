# Anvilite v0.1 - 使用逻辑文档

## 一、产品定位

Anvilite 是一款游戏化个人生产力桌面应用。用户通过完成现实中的任务和习惯，获得经验值（XP）与矿石（Ore），驱动角色升级、区域繁荣、徽章解锁等成长体系，将日常效率管理转化为可视化的冒险历程。

---

## 二、核心循环

```
完成任务/习惯 → 获得 XP + 矿石
       ↓
  角色升级 → 解锁称号
       ↓
区域繁荣提升 → 解锁装饰商店商品
       ↓
  成就徽章 → 里程碑殿堂铭刻
       ↓
时光卷轴记录 → 淬火重铸（Prestige）
```

---

## 三、功能模块

### 3.1 仪表盘（Dashboard）

应用主页，采用 24 列可拖拽网格布局，包含以下卡片：

| 卡片 | 功能 |
|------|------|
| 任务列表 | 今日待办，按分类/优先级筛选 |
| 习惯追踪 | 今日到期习惯，一键完成/跳过/缺席 |
| 角色卡片 | 等级、XP 进度条、矿石余额、全局状态 |
| 统计面板 | 今日完成数、累计 XP |
| 快速创建 | 一行输入快速添加任务 |
| 成长趋势 | 完成热力图日历 |

### 3.2 任务系统（Tasks）

- **创建**：标题、分类、难度（1-5星）、优先级（紧急/高/中/低）、截止日期、预估时间
- **执行**：开始计时 → 暂停 → 完成（自动累计实际用时）
- **完成**：获得 XP 和矿石，高难度（4-5星）需二次确认
- **撤销**：已完成任务可撤销，XP 回收
- **删除**：软删除 → 回收站 → 永久删除
- **隐藏**：不想看到但不想删除的任务

**XP 计算**：
- 基础 XP = 难度映射（1→1, 2→2, 3→3, 4→5, 5→8）
- 加成 = 按时完成 +20% + 连续活跃天数奖励（3天+10%, 7天+20%, 14天+30%, 30天+50%）+ 高难度 +50%
- 矿石 = XP 的 1:1 比例

### 3.3 习惯系统（Habits）

**重复模式**：
| 模式 | 说明 |
|------|------|
| 每天 | 每日出现 |
| 工作日 | 周一至周五 |
| 每周（严格） | 指定周几出现 |
| 每周（弹性） | 每周完成 N 次，自由安排 |
| 隔周 | 每 14 天一次 |
| 每月 | 每月一次 |
| 每月固定 | 指定每月几号（支持月末） |
| 自定义 | 每 N 天一次 |

**连续机制**：
- 完成 → 连续次数 +1
- 跳过 → 连续次数 -10%（下取整）
- 缺席 → 连续次数 -50%，若有容错充能则抵消
- 容错充能：首次连续 7 次获得 1 次，之后每 14 次获得 1 次（上限 1）

**状态流转**：
```
active → completed_today（当日完成，次日自动恢复 active）
active → paused（暂停）→ active（恢复）
active → mastered（精通标记）
active → archived/hidden（归档/隐藏）
```

### 3.4 世界地图（World Map）

- 画布上最多放置 12 个区域节点，每个区域绑定一个分类（如"工作""学习"）
- 支持平移、缩放、拖拽定位区域
- 编辑模式下可重命名（限 10 字）、删除区域（需先清空关联任务/习惯）
- 每个区域有 6 级繁荣度：荒芜 → 聚落 → 丰饶 → 繁荣 → 鼎盛 → 辉煌
- 繁荣度由该分类下任务累计 XP 驱动，对数增长

**12 个区域模板**：
家园(home)、竞技场(arena)、图书馆(library)、工坊(workshop)、锻造厂(forge)、里程碑殿堂(milestone)、温泉(spring)、议事厅(council)、远征地(expedition)、天文台(observatory)、花园(garden)、广场(plaza)

### 3.5 区域内部（Interior Space）

- 点击区域进入内部视图，展示繁荣度详情
- 显示该分类下的任务和习惯列表
- **装饰商店**：用矿石购买装饰物（每区域 6 种，分 6 级解锁，价格 30~1500 矿石）
- 里程碑殿堂（特殊区域）展示成就徽章、铭刻记录

### 3.6 里程碑殿堂（Milestone Hall）

三栏布局：
1. **左栏**：角色头像、等级信息、能力图谱雷达图
2. **中栏**：徽章陈列柜（31+ 枚静态徽章 + 动态区域繁荣徽章）
3. **右栏**：时光卷轴（铭刻的里程碑事件流）

**徽章类别**：起步、等级、坚持（活跃天数）、习惯（单习惯连续）、任务量、投入时间、收集（主题）、淬火（重铸）、区域繁荣

### 3.7 时光卷轴（Timeline）

按时间倒序记录所有成长事件：
- 任务完成、习惯完成/跳过/缺席
- 升级、获得徽章、区域繁荣升级
- 淬火重铸
- 用户手动铭刻的里程碑

### 3.8 角色系统

- **等级**：XP 公式 `5 × ln(level+1) × level`，无上限
- **称号**：锻造系 / RPG 系 / 自定义，按等级阶梯自动切换
- **全局状态**：活跃 / 充能 / 休息 / 旅行 / 归来（点击切换，纯装饰）
- **淬火重铸（Prestige）**：Lv.51 后可重铸，等级归 1，保留所有数据，记录重铸次数

### 3.9 设置

- **账号管理**：创建/切换/删除本地账号，数据完全隔离
- **角色名**：自定义显示名称
- **主题**：8 种主题（4 亮色 + 4 暗色），部分需矿石解锁（200~500 矿石）
- **语言**：中文 / English

---

## 四、数据与存档

- 所有数据存储在本地 localStorage
- 每个账号独立存储，key 前缀隔离（默认账号 `anvilite-*`，新账号 `anvilite-{uuid}-*`）
- 切换账号时应用自动重载
- 无云同步，数据完全离线
