# Changelog

## v0.3.0 — 2026-04-14

### 新功能

**流程模块（Flows）**
- 新增第三个主 Tab（任务 → 流程 → 世界地图 → 里程碑殿堂）
- 4 种流程类型：流程型（有序编号步骤）、检查型（逐项确认）、清单型（无序列表）、日程型（时间轴）
- 流程型、检查型、清单型支持「开始执行」模式，逐步打勾、实时进度显示，退出自动重置
- 「转为任务」：将步骤一键转为任务子任务，支持自定义标题/分类/难度/优先级/截止日期
- 内置 8 个系统模板（不可编辑），可「复制为我的流程」后修改
- 文件夹管理：新建/重命名/删除文件夹，折叠状态持久化
- 仪表盘新增「我的流程」卡片，点击条目直接跳转

**子任务 / 子习惯折叠交互**
- 任务和习惯子项新增折叠按钮（勾选框右侧 `▾/▸ N/M`）
- 折叠状态跨页面持久化（uiStore，Zustand persist）
- 折叠逻辑统一覆盖：任务列表、仪表盘任务卡片、任务列表习惯 Tab、仪表盘习惯卡片
- 子项的子项同样支持独立折叠，最多 3 层嵌套

**仪表盘增强**
- 任务卡片移除 10 条上限，今日待办全量显示
- 任务卡片和习惯卡片均显示子项（compact 模式）
- 仪表盘任务切换状态时无飞出动画，改为淡出（0.15s）

**世界地图改造**
- 响应式卡片网格：`computeOptimalCols` 算法实时计算最优列数，卡片面积最大化
- 编辑模式改为单列拖拽排序（原网格多列拖拽兼容性问题已解决）
- 修复：区域内部空间返回后卡片尺寸异常（ResizeObserver 绑定丢失）
- 修复：切换 Tab 时出现缩放动画（移除 AreaCard 的 `layout` 属性）

**灵感速记**
- 仪表盘新增灵感卡片，支持拖拽排序
- `Ctrl+Shift+N` 全局唤起输入框，`Ctrl+Enter` 快速保存

### 功能增强

- **任务/习惯卡片 hover 延迟**：操作栏延迟 300ms 显示，避免鼠标扫过时频繁弹出
- **仪表盘热力图响应式**：格子大小随卡片尺寸自适应，月视图/周视图均支持
- **热力图周视图修复**：时段数据基于 GrowthEvent 真实小时统计（原均分算法已修复）
- **编辑抽屉实际用时**：实际时长输入框真正可编辑（原 React 受控 bug 已修复）
- **Electron 图标**：打包后任务栏图标正确显示

### 数据

- 导出备份新增 `sops`（流程）和 `inspirations`（灵感）字段
- v0.2 备份文件可兼容导入（流程/灵感数据为空）
- `APP_VERSION` 更新为 `0.3.0`

### 技术

- 新增测试覆盖：`systemSOPs`（17 cases）、`formatRelativeTime`（12 cases）、`computeOptimalCols`（11 cases）、`uiStore`（9 cases），共 169 tests
- 新增 `src/utils/formatRelativeTime.ts`、`src/utils/grid.ts`（纯函数提取，便于单元测试）
- 新增 `src/test/setup.ts`（Vitest 全局 localStorage polyfill）
- `vitest.config.ts` 新增 `setupFiles` 配置

## v0.2.0 — 2026-04-05

### 新功能

**灵感速记（NEW-01）**
- `Ctrl+Shift+N` 随时唤起灵感输入浮窗，`Ctrl+Enter` 快速保存
- 仪表盘新增灵感卡片，展示所有未转化灵感
- 一键将灵感转为任务（预填标题和分类）
- 支持灵感内容内联编辑

**档案馆模块（PATCH-04）**
- 世界地图档案馆区域内部新增双 Tab 布局
- 数据总览：5 项终身统计（已完成任务、习惯完成次数、累计 XP、最长连续、分类 XP 雷达图）
- 时光卷轴迁移至档案馆，卡片上下交替布局，悬停展开详情

**连击阶梯跃升动画（PATCH-05）**
- 连续活跃天数跨越 3 / 7 / 14 天：浮动徽章弹出动画
- 跨越 30 天：全屏庆祝特效 + 粒子爆炸

**已完成项分组视图（FEAT-06/07）**
- 任务列表支持按月份、按区域、自定义三种方式归档已完成任务
- 习惯支持按区域、自定义分组查看已完成/归档记录
- 自定义分组支持拖拽将任务移入/移出

**习惯完成撤销（FEAT-09）**
- 完成习惯后，当天内可通过仪表盘习惯卡片的撤销按钮取消本次完成

**仪表盘已完成项优化（FEAT-08）**
- 仪表盘任务卡片仅展示今日完成的任务（不累积历史已完成项）
- 习惯卡片展示当前周期内已完成的习惯

**存储空间监控**
- 设置页面显示 localStorage 用量进度条
- 用量超过 80% 时仪表盘顶部显示警告，提供"立即导出"快捷入口
- 用量超过 95% 时自动清理最旧的成长事件（保留最新 500 条）

### 功能增强

**角色状态自动检测（FEAT-03）**
- 有进行中任务时，角色状态自动切换为对应分类的工作状态
- 无进行中任务时，恢复为用户手动选择的状态

**任务拖拽排序（FEAT-01）**
- 任务列表支持在同分组内拖拽调整顺序
- 排序结果持久化保存

**编辑抽屉补全实际用时（BUG-06）**
- 已完成任务的编辑抽屉中新增「实际用时」输入框
- 未完成任务显示灰色占位提示

**跨午夜习惯状态自动刷新（BUG-08）**
- 应用长期运行不重启时，次日 0 点自动将已完成习惯恢复为 active
- 从后台切回前台时立即检查日期变化

**数据导出与导入**
- 设置页面支持将全部数据导出为 JSON 备份文件
- 支持从 JSON 备份文件导入恢复数据

## v0.1.0 — 2026-03-28

首个可运行版本，包含核心功能：

- 任务 CRUD（含子任务、优先级、截止日期、难度、计时）
- 习惯系统（含多种周期、连续机制、容错充能）
- XP 与等级体系（对数曲线、矿石双货币、称号系统）
- 仪表盘（可拖拽卡片网格、热力图）
- 世界地图（12 区域、6 级繁荣度、进入动画）
- 区域内部空间（任务/习惯/装饰/统计）
- 里程碑殿堂（徽章 31+枚、能力雷达图）
- 时光卷轴（全量成长事件记录）
- 8 套主题（4 亮 + 4 暗）
- 中英双语 i18n
- 多账号本地隔离存储
