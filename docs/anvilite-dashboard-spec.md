# Anvilite · 锻石 — 仪表盘模块规格文档

> **版本**: v1.1
> **日期**: 2026-03-27
> **状态**: 已锁定
> **变更记录**: v1.1 角色迷你卡新增计时显示、区域名称更新、习惯卡片刷新时间显示

---

## 一、模块定位

仪表盘是用户打开Anvilite后的**默认页面**，回答用户的核心问题：「我该做什么，我做得怎么样了？」

---

## 二、整体布局

### 2.1 布局类型

**卡片式自由布局**，用户可拖拽卡片位置、调整卡片大小（完全自由）。

### 2.2 默认布局（新用户）

```
┌─────────────────────────────────────────────────────┐
│  早上好，[角色名]                                      │
├──────────────────────────────┬──────────────────────┤
│                              │  [今日统计卡片]        │
│                              ├──────────────────────┤
│                              │  [角色迷你状态卡]       │
│    [任务卡片]                  ├──────────────────────┤
│    占左侧 2/3                 │  [习惯卡片]           │
│    显示所有已添加任务            ├──────────────────────┤
│                              │  [快速创建卡片]        │
│                              ├──────────────────────┤
│                              │  [成长趋势卡片]        │
├──────────────────────────────┴──────────────────────┤
```

### 2.3 卡片管理

- 可拖拽任意卡片改变位置
- 可拖拽边缘调整大小
- 可隐藏任意卡片，可在编辑模式中添加回来
- 布局持久化保存

## 三、问候语区域

固定顶部，不可拖拽。不显示额外日期信息。

| 时段 | 时间范围 | 中文 | 英文 |
|------|---------|------|------|
| 早上 | 5:00 - 11:59 | 早上好，[角色名] | Good morning, [name] |
| 下午 | 12:00 - 17:59 | 下午好，[角色名] | Good afternoon, [name] |
| 晚上 | 18:00 - 4:59 | 晚上好，[角色名] | Good evening, [name] |

样式：H2(20px)，weight 700，角色名使用accent色。

## 四、卡片定义

### 4.1 任务卡片

**内容**：显示**所有已添加的任务**（不限于今日），与任务Tab共享数据源。

**与任务Tab的区别**：
- 信息简化：只显示 **勾选框 + 标题 + 优先级 + 截止日期**
- 进行中任务额外显示**计时器** ★v1.1新增
- 无分类Tab筛选栏
- 无快速创建输入框（独立为单独卡片）
- 其余交互一致：勾选完成 → 飞出动画 → Toast撤销

**卡片内信息布局**：

普通任务：
```
[勾选框] [标题] ............... [优先级] [截止日期]
```

进行中任务 ★v1.1新增：
```
[■停止] [标题] ................. [优先级] [⏱ 00:32:15]
```

**分组**：进行中 → 待办 → 已完成（不折叠，带撤销按钮）

**默认尺寸**：左侧约2/3宽度。

### 4.2 习惯卡片

**内容**：只显示**当前周期内未完成**的习惯。

**显示规则**：
- 每日习惯：今天未完成的显示
- 工作日习惯：今天是工作日且未完成的显示
- 每周习惯（严格）：今天是指定日且未完成的显示
- 每周习惯（灵活）：本周内还未达到总次数的显示
- 其他周期：当前周期内未完成的显示
- 已完成/已暂停不显示

**非每日习惯的刷新时间** ★v1.1明确：
```
[完成] 每周跑步3次     🔥5连续  [跳过]
       下次刷新：周三           ← dim色 Tiny字号(10px)
```

**卡片内信息布局**：
```
[完成按钮] [习惯名称] ........... [🔥X连续] [跳过按钮(淡化)]
                                  [下次：周三 (仅非每日)]
```

**操作**：
- 完成 → 获得XP+矿石，触发反馈动画，本周期内消失
- 跳过 → 连续-10%，本周期内消失

**空状态**：「今天的习惯都完成了！」或「还没有习惯，去区域内部添加一个吧」

### 4.3 今日统计卡片

三个统计指标横排：

```
┌──────────┬──────────┬──────────┐
│   X 个    │   X XP   │  🔥 X天  │
│  今日完成  │  今日获得  │  连续活跃  │
└──────────┴──────────┴──────────┘
```

- **今日完成**：今天完成的任务+习惯总和
- **今日获得**：今天获得的总XP
- **连续活跃**：当前连续天数，配🔥图标
- 数字：H2(20px)，weight 800，数字字体规则
- 标签：Small(12px)，dim色

### 4.4 角色迷你状态卡 ★v1.1更新

```
┌─────────────────────────────┐
│  [状态图标] [状态文字]         │ ← 📚 学习中 · 32分钟
│  Lv.12 · 锤炼                │
│  ████████░░░░ 45/208 XP      │
└─────────────────────────────┘
```

**当前状态的判定逻辑** ★v1.1更新（使用新区域名称）：

有「进行中」任务时，按任务所属分类显示：

| 分类（区域） | 图标+文字 |
|------------|---------|
| 书阁高塔（学习） | 📚 学习中 |
| 锻造坊（工作） | 💼 工作中 |
| 竞技场（运动） | 🏃 运动中 |
| 灵感工坊（兴趣） | ✨ 创作中 |
| 家园（生活） | 🏠 生活中 |
| 自定义区域 | 📌 忙碌中 |

有进行中任务时，状态文字后追加计时时长 ★v1.1新增：
- 格式：「📚 学习中 · 32分钟」
- 分钟数取自当前进行中任务的计时器
- 多个进行中任务时显示优先级最高的那个

无「进行中」任务时，显示角色基础状态：

| 状态 | 图标+文字 |
|------|---------|
| active | ⚡ 就绪 |
| charging | 🔋 充电中 |
| resting | 💤 休憩中 |
| traveling | 🌄 远行中 |
| returning | 🌅 归来 |

**等级和称号**：`Lv.X · 称号名`，accent色+数字字体
**经验条**：4px高，accent色填充，右侧显示 `当前/所需 XP`

**点击**：可跳转到里程碑殿堂查看完整角色信息。

### 4.5 快速创建卡片

独立卡片，在右侧。

```
┌─────────────────────────┐
│  + 快速创建任务...         │ ← dashed边框输入框
│                          │
│  [家园] [竞技场] [书阁高塔] │ ← 点击输入框后展开分类快选
│  [灵感工坊] [锻造坊] [其他▼]│
└─────────────────────────┘
```

- 默认只显示输入框，点击后展开分类按钮
- 分类按钮组：预设区域直接显示，自定义区域在「其他▼」下拉中
- 回车创建，不选分类则用默认分类

### 4.6 成长趋势卡片

分为两部分：**数据摘要 + 热力图**。

**数据摘要**：
```
┌──────────────────────────────┐
│  本周         │  本月          │ ← Tab切换
├──────────────────────────────┤
│  完成任务 12个  坚持习惯 5个    │
├──────────────────────────────┤
│  [热力图]                     │
└──────────────────────────────┘
```

**热力图 — 月视图**（选「本月」时）：
- 精度：**每天**
- 布局：GitHub贡献图式，7行(周一~日) × 4-5列(每周)
- 颜色深浅 = 当天获取XP总量
- 颜色跟随主题accent色变化
- 阈值根据用户近30天日均XP动态计算

**热力图 — 周视图**（选「本周」时）：
- 精度：**每天4个时段**
- 布局：7列(周一~日) × 4行(4时段)
- 时段划分：
  - 行1：1:00-6:00（凌晨）
  - 行2：6:00-12:00（上午）
  - 行3：12:00-18:00（下午）
  - 行4：18:00-24:00（晚上）
- 左侧时段标签，底部星期标签

**热力图通用规则**：
- 格子：正方形，圆角sm(4px)，间距2px，大小16-20px自适应
- 颜色层级：无XP=border色，低=accent 20%，中=accent 50%，高=accent 80%，极高=accent 100%
- 悬停tooltip：「周三下午 · +12 XP · 完成3个任务」

## 五、边界情况

### 5.1 新用户

- 任务卡片：空状态插画 + 「创建你的第一个任务吧」
- 习惯卡片：「去区域内部添加你的第一个习惯」
- 统计：全部显示0
- 角色迷你卡：Lv.1 · 破壳，⚡ 就绪
- 热力图全空
- 快速创建卡片正常显示

### 5.2 全部任务完成

- 任务卡片显示已完成列表 + 「所有任务都完成了！」
- 统计数字正常显示今日成果

### 5.3 长时间未使用后回来

- 问候语正常（不提及离开时长）
- 角色迷你卡显示当前状态（充电中/休憩/远行）
- 连续天数显示0或1
- 热力图中缺失天数显示为border色

### 5.4 窗口大小变化

- 卡片响应式：窗口缩小时自动换行
- 最小宽度下所有卡片单栏纵向排列
- 热力图格子随卡片宽度自适应

## 六、与其他模块的关系

| 交互 | 说明 |
|------|------|
| 任务卡片 → 任务Tab | 数据共享，操作同步 |
| 任务卡片 → 详情抽屉 | 点击打开右侧抽屉 |
| 习惯卡片 → 区域内部空间 | 数据来源，完成操作同步 |
| 角色迷你卡 → 里程碑殿堂 | 点击跳转 |
| 角色迷你卡 → 任务计时器 | 显示当前计时中的任务分类和时长 ★v1.1 |
| 统计卡片 → XP/矿石系统 | 数据来源 |
| 成长趋势 → GrowthEvent | 热力图数据来源 |
| 快速创建 → 任务系统 | 创建任务同步 |
