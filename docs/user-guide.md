# Anvilite · 锻石 — 用户指南

> 当前版本：v0.2

---

## 一、产品定位

Anvilite 是一款游戏化个人生产力桌面应用。用户通过完成现实中的任务和习惯，获得经验值（XP）与矿石（Ore），驱动角色升级、区域繁荣、徽章解锁等成长体系。

**核心理念**：角色是你真实成长的镜像，不是脱离现实的游戏壳。没有惩罚机制，只有向前的动力。

---

## 二、核心循环

```
完成任务 / 习惯
       ↓
获得 XP + 矿石
       ↓
角色升级 → 解锁称号
区域繁荣 → 解锁装饰
       ↓
徽章获得 → 档案馆铭刻
时光卷轴记录完整历程
```

---

## 三、功能模块

### 3.1 仪表盘（Dashboard）

应用主页，可拖拽、调整大小、显示/隐藏的卡片布局。

| 卡片 | 功能 |
|------|------|
| 任务列表 | 今日任务（todo / doing），按分类/优先级筛选，拖拽排序 |
| 习惯追踪 | 今日到期习惯，点击直接完成，含当日撤销按钮 |
| 角色卡片 | 等级、XP 进度条、矿石余额、全局状态 |
| 统计面板 | 今日完成数、今日获得 XP、当前连续天数 |
| 快速创建 | 一行输入快速添加任务（Enter 确认）|
| 成长热力图 | 完成热力图日历，月视图/周视图可切换 |
| 灵感记录 | 未转化的灵感列表，支持快速转为任务 |

**灵感速记快捷键**：`Ctrl+Shift+N` 随时唤起灵感输入框，`Ctrl+Enter` 快速保存。

### 3.2 任务系统（Tasks）

**创建**

- 顶部输入框快速创建（回车确认）
- 右侧抽屉完整编辑：标题、分类、难度（1-5 ★）、优先级（紧急 / 高 / 中 / 低）、截止日期、预估时长、备注
- 最后填写的实际用时在任务完成后可在抽屉中补录

**子任务**

- 在编辑抽屉中添加子项，最多 3 层嵌套（根 → 子 → 孙）
- 子任务继承父任务的分类和难度
- 子任务完成不单独计算 XP，仅显示完成动效
- 删除父任务会级联删除所有子孙任务

**完成与撤销**

- 勾选框完成，4-5 ★ 高难度需二次确认
- 完成后 5 秒内可通过 Toast 撤销；已完成列表也可随时撤销
- 撤销时 XP 收回（等级允许降低），矿石永久保留

**排序与筛选**

- 状态自动分组：进行中 → 待办 → 已完成
- 拖拽可手动调整同分组内的任务顺序
- 顶部分类 Tab 筛选任务

**已完成任务分组视图**

- 支持按月份、按区域分类、自定义分组三种方式归档完成的任务
- 自定义分组支持拖拽将任务移入/移出分组

**XP 计算**（仅根任务）

| 要素 | 加成 |
|------|------|
| 基础 XP | ★=1, ★★=2, ★★★=3, ★★★★=5, ★★★★★=8 |
| 按时完成 | +20% |
| 连续活跃天数 | +3%/天，上限 +50%（约 17 天达到上限）|
| 高难度（4-5 ★） | +50% |

矿石 = XP 的 1:1 同步获得。

### 3.3 习惯系统（Habits）

**创建**：名称、分类、难度、重复周期（每天 / 工作日 / 每周 / 隔周 / 每月 / 自定义），在区域内部空间或任务列表的习惯 Tab 中管理。

**重复模式**

| 模式 | 说明 |
|------|------|
| 每天 | 每日出现 |
| 工作日 | 周一至周五 |
| 每周（严格）| 指定周几出现 |
| 每周（弹性）| 每周完成 N 次，自由安排 |
| 隔周 | 每 14 天一次 |
| 每月 | 每月一次 |
| 自定义 | 每 N 天一次 |

**连续机制**

- 完成 → 连续次数 +1
- 跳过 → 连续次数 -10%（下取整）
- 缺席 → 连续次数 -50%，若有容错充能则抵消
- 容错充能：首次连续 7 次获得 1 次，之后每 14 次获得 1 次（上限持有 1 次）

**当日撤销**：完成习惯后，当天内可通过仪表盘习惯卡片右侧的撤销按钮取消本次完成。

**已完成习惯分组视图**：支持按区域、自定义分组查看已完成/归档的习惯。

**状态流转**
```
active → completed_today（次日 0 点自动恢复 active）
active ⇄ paused（暂停 / 恢复）
active → archived（完成并归档）
active → hidden（隐藏）
```

### 3.4 世界地图（World Map）

- 固定画布，自动缩放适配窗口，可滑动查看
- 最多放置 12 个区域节点，每个区域绑定一个分类
- 编辑模式：拖拽调整节点位置、重命名（限 10 字）、删除区域
- 单击区域：查看底部信息栏（名称、繁荣等级、任务数、XP、进入按钮）
- 双击区域：进入区域内部空间

**繁荣度等级**

| 等级 | 名称 |
|------|------|
| 1 | 荒芜 |
| 2 | 聚落 |
| 3 | 丰饶 |
| 4 | 繁荣 |
| 5 | 鼎盛 |
| 6 | 辉煌 |

繁荣度由该区域分类下任务/习惯累计 XP 驱动（对数增长）。

**12 个区域模板**：家园、竞技场、书阁高塔、灵感工坊、锻造坊、档案馆、温泉、议事厅、远征大厅、天文台、植物园、广场

### 3.5 区域内部空间（Interior）

双击进入任意区域，看到以下内容：

- **左侧面板**：该区域的任务列表 + 习惯列表（含管理入口）
- **右侧面板**：技能等级与进度、XP / 完成数 / 投入时长 / 繁荣等级统计
- **装饰商店**：用矿石购买该区域专属装饰（随繁荣等级解锁）

按 `ESC` 或点击左上角「← 返回地图」可退出。

### 3.6 档案馆（Archive Hall）

特殊区域（世界地图上固定位置），内含两个 Tab：

**数据总览**

| 统计项 | 说明 |
|--------|------|
| 已完成任务 | 全部完成过的根任务数量 |
| 习惯完成次数 | 全部习惯累计完成次数 |
| 累计经验 | 角色历史总获得 XP |
| 最长连续 | 历史最长活跃天数连击 |
| 分类 XP 分布 | 各区域技能雷达图 |

**时光卷轴**

按时间倒序记录所有成长事件：
- 任务完成、习惯完成
- 角色升级、获得徽章
- 区域繁荣度升级
- 用户手动铭刻的里程碑

所有事件永久保留，不自动清理。

### 3.7 角色系统

- **等级**：XP 公式 `5 × ln(level+1) × level`，无上限
- **等级允许降级**：撤销任务/习惯时 XP 减少，等级随之真实降低
- **称号**：按等级自动切换，两套预设（锻造系 / 经典 RPG，各 8 级）+ 完全自定义
- **全局状态**：有进行中任务时自动显示对应工作状态；无任务时可手动切换（活跃 / 充电 / 休憩 / 远行 / 归来）
- **淬火重铸（Prestige）**：Lv.51 后可重铸，等级归 1，保留所有数据和矿石，记录重铸次数

### 3.8 成就与徽章

31+ 枚静态徽章 + 动态区域繁荣徽章，分 8 类：
- **起步**：首次完成任务 / 习惯
- **等级**：达到特定等级
- **坚持**：连续活跃天数（3 / 7 / 14 / 30 天等）
- **习惯**：单习惯连续完成次数
- **任务量**：累计完成任务数
- **投入时间**：累计计时总时长
- **收集**：解锁特定数量主题
- **区域繁荣**：各区域达到指定繁荣等级

徽章获得时弹出通知，可在档案馆的时光卷轴中查看历史。

### 3.9 视觉反馈

| 事件 | 反馈 |
|------|------|
| 完成任务 | XP 飘字动画（右侧浮现） |
| 角色升级 | 全屏庆祝 + 粒子爆炸效果 |
| 连击里程碑（3/7/14天） | 浮动徽章动画 |
| 连击里程碑（30天） | 全屏庆祝特效 |
| 获得新徽章 | 右上角通知 |
| 操作撤销 | 底部 Toast 提示 |

### 3.10 设置

- **账号管理**：创建 / 切换 / 删除本地账号，数据完全隔离
- **角色名**：自定义显示名称
- **主题**：8 种主题（4 亮色 + 4 暗色），部分需矿石解锁
  - 亮色：晨光白（默认，免费）、薄荷纸、羊皮卷、云石灰
  - 暗色：锻铁紫、矿石青、烈焰铸、翠林绿
- **语言**：中文 / English
- **存储空间**：显示 localStorage 用量进度条，接近上限时仪表盘顶部显示警告

---

## 四、数据与存档

- 所有数据存储在本地 **localStorage**（无云同步，完全离线）
- 每个账号独立存储，key 前缀隔离
- 支持**数据导出**（JSON 格式备份）和**数据导入**（从备份恢复）
- 入口：设置 → 数据管理
