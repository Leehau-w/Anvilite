# Anvilite · 锻石 — 区域内部空间模块规格文档

> **版本**: v1.0
> **日期**: 2026-03-27
> **状态**: 已锁定

---

## 一、模块定位

区域内部空间是用户从世界地图zoom进入后的沉浸式场景。像素场景全屏展示，功能面板以半透明卡片覆盖其上。这里是用户在某个特定领域（学习/工作/运动等）的"专属工作室"。

---

## 二、整体布局

### 2.1 场景层（底层）

- **全屏展示**等距像素场景，占满整个内容区域
- 每个区域有**独特场景**（书房/训练场/工作室等）
- 场景**随繁荣度变化**（与世界地图上的6级对应，但视角是内部视角）
- 角色在场景中**固定位置**展示

### 2.2 功能面板层（上层）

- **默认隐藏**，场景完全可见
- 通过场景上的**按钮触发**展开功能面板
- 面板以**半透明+微弱blur+边框**（磨砂玻璃效果）覆盖在场景上
- 面板展开后场景仍然透过面板可见

## 三、半透明面板样式

### 3.1 磨砂玻璃效果

```css
.glass-panel {
  background: rgba(var(--color-surface-rgb), 0.75);  /* 75%透明度 */
  backdrop-filter: blur(8px);
  border: 1px solid rgba(var(--color-border-rgb), 0.5);
  border-radius: var(--radius-xl);  /* 16px */
  padding: var(--spacing-lg);  /* 16px */

```

- 暗色主题：背景更深（rgba 0.80），blur更强（12px）
- 亮色主题：背景更浅（rgba 0.70），blur更轻（6px）

### 3.2 展开/收起按钮

- **位置**：右上角（与「返回地图」按钮同一行）
- **收起状态**：显示图标按钮「☰ 面板」
- **展开状态**：显示图标按钮「✕ 收起」
- 按钮样式：Ghost按钮 sm尺寸，半透明背景
- 展开动画：面板从两侧滑入，300ms ease-out
- 收起动画：面板向两侧滑出，250ms ease-in

## 四、角色在场景中的行为

### 4.1 角色位置

- 固定在场景的中央偏下位置
- 功能面板展开时，角色在左右面板之间的空隙处可见
- 角色不可交互（不可点击/拖拽）

### 4.2 任务进行中状态

当该区域有任务处于「进行中」状态（计时中）时：

| 区域 | 角色工作动作 | 场景氛围 |
|------|------------|---------|
| 家园 | 整理、烹饪、园艺 | 炊烟、暖光 |
| 竞技场 | 挥剑、跑步、格斗 | 沙尘飞扬、火把闪烁 |
| 书阁高塔 | 翻书、写字、冥想 | 翻页粒子、油灯摇曳 |
| 灵感工坊 | 绘画、雕刻、弹琴 | 音符粒子、颜料飞溅 |
| 锻造坊 | 打铁、操作齿轮 | 火花飞溅、蒸汽上升 |
| 里程碑殿堂 | 审视展品 | 光柱缓缓移动 |
| 自定义模板区域 | 各区域对应的工作动作 | 各区域对应的动态效果 |
| 空白自定义区域 | 通用忙碌动作 | 无特殊效果 |

### 4.3 空闲状态

当该区域没有任务处于「进行中」时：

- 角色在场景中站立/散步/休息（轻微的idle动画）
- 场景氛围安静平和（无动态粒子效果）
- 如果角色全局状态是充电中/休憩/远行，则显示对应的休息状态

### 4.4 角色状态与场景的关系

- 角色在**哪个区域的内部空间**中显示工作状态，取决于**哪个区域有进行中的任务**
- 如果用户在书阁高塔的内部空间，但进行中的任务属于锻造坊，那书阁高塔里的角色显示**空闲状态**
- 角色的工作/空闲只反映**当前所在区域**的任务状态

## 五、任务计时器（新增功能）

### 5.1 概述

当用户将任务状态从「待办」切换为「进行中」时，自动开始计时。任务完成时停止计时。计时时长即为该任务的实际投入时长。

### 5.2 计时规则

- **开始计时**：任务状态变为「进行中」的瞬间
- **停止计时**：任务状态变为「已完成」或切回「待办」的瞬间
- **暂停计时**：关闭应用时自动暂停，重新打开后如果任务仍为进行中则继续计时
- **多任务**：允许多个任务同时进行中（同时计时）
- **计时数据**：记录在任务的 `actualMinutes` 字段
- **显示位置**：
  - 任务详情抽屉中显示实时计时「已进行 32分钟」
  - 仪表盘角色迷你卡：「📚 学习中 · 32分钟」
  - 区域内部数据面板：累计投入时长

### 5.3 计时器UI

在任务列表中，进行中的任务卡片上显示一个小计时器：

```
[■ 停止] 完成日语N3语法笔记      紧急  今天  ⏱ 00:32:15
```

- ⏱ 图标 + 实时跳动的时间（HH:MM:SS）
- 勾选框变为「■ 停止」按钮（点击可暂停回到待办状态）
- 计时器使用数字字体规则（Inter Semi Bold, tabular-nums）

### 5.4 对任务模块的影响（回填）

任务数据结构新增字段：
```typescript
actualMinutes: number;        // 实际投入分钟数（由计时器累计）
timerStartedAt: string | null; // 计时开始时间（进行中时有值）
```

## 八、矿石与装饰商店

### 8.1 矿石（Ore）系统

**获取方式**：
- 每次完成任务/习惯时，与XP同时获得
- 获取数量与XP **1:1**（获得5XP同时获得5矿石）
- 加成规则也与XP一致（按时/连击/高难加成同样适用）

**消费方式**：
- 购买区域装饰
- 解锁主题皮肤

**不可消费的场景**：
- 矿石不会因撤销/删除任务而收回（只有XP会收回）
- 矿石只增不减（除了购买消费）

**显示位置**：
- 区域内部右侧面板（收藏区域标题旁）
- 底部状态栏可选显示
- 装饰商店内

**数据结构**：
```typescript
ore: number;  // 存储在角色数据中，当前矿石余额
totalOreEarned: number;  // 历史累计获得（不减少）
```

### 8.2 装饰商店

**入口**：区域内部右侧面板的「装饰商店」按钮

**展示方式**：弹窗/模态框

**商店内容**：
- 按繁荣等级分组展示可购买的装饰
- 每个繁荣等级解锁一批新装饰
- 低于当前繁荣等级的装饰可购买
- 高于当前繁荣等级的装饰显示为锁定（灰色，显示所需等级）

**装饰物品卡片**：
```
┌──────────┐
│ [缩略图]  │
│  书架     │
│  ⛏ 15    │
│ [购买]    │
└──────────┘
```

**购买流程**：
1. 点击「购买」
2. 确认弹窗：「花费 15 矿石购买 [书架]？当前余额：42」
3. 确认后扣除矿石，装饰加入已拥有列表
4. 装饰自动出现在区域内部场景中

**每个区域的装饰独立**：
- 书阁高塔的装饰是书架、卷轴、星象仪等
- 竞技场的装饰是奖杯、旗帜、训练器械等
- 装饰只出现在对应区域的内部场景中

### 8.3 主题皮肤解锁

- 8套主题中，「晨光白」为默认免费
- 其余7套可通过矿石解锁
- 解锁价格待定（建议根据使用频率和总矿石产出量平衡）
- 主题皮肤入口在设置中，已解锁的直接切换，未解锁的显示价格和「解锁」按钮

## 九、区域内部场景的繁荣度变化

每个区域的内部场景有6级视觉变化，与世界地图上的繁荣度同步：

### 9.1 场景变化原则

- 内部场景是世界地图建筑的**内部视角**
- 荒芜级：破败、空旷、昏暗
- 随繁荣度提升：空间变大、家具增多、光线变亮、氛围变好
- 辉煌级：华丽、充满魔法/锻石元素
- 用户购买的装饰会**叠加**在场景上

### 9.2 各区域内部场景概述

| 区域 | 荒芜 | 辉煌 |
|------|------|------|
| 家园 | 破旧帐篷内部，泥地 | 奇迹城堡大厅，锻石微光 |
| 竞技场 | 露天泥地，断木 | 巨型斗兽场内部，战神巨像 |
| 书阁高塔 | 破损石塔底层，漏风 | 宏伟学院穹顶，智慧星图 |
| 灵感工坊 | 灰尘木屋，锈蚀工具 | 超现实创意空间，灵感繁星 |
| 锻造坊 | 冰冷残炉，锈蚀铁砧 | 锻石幽光炉火，现实波纹 |
| 里程碑殿堂 | 空旷暗厅 | 英灵殿，悬浮锻石核心 |

自定义模板区域参照世界地图文档中的描述。

## 十、导航与操作

### 10.1 返回地图

- **位置**：左上角固定
- **按钮**：Ghost按钮 sm尺寸，「← 返回地图」
- **半透明背景**确保在任何场景上都可读
- **ESC键**也可触发返回
- **动画**：zoom拉远800ms（与进入动画反向）

### 10.2 面板内的详情操作

- 点击任务卡片 → 右侧详情抽屉（与任务Tab一致）
- 点击「+ 创建任务」→ 右侧抽屉（分类自动填入当前区域）
- 点击「+ 创建习惯」→ 右侧抽屉（分类自动填入当前区域）
- 点击「管理习惯」→ 右侧抽屉（习惯管理视图）
- 点击「装饰商店」→ 模态弹窗

### 10.3 详情抽屉与面板的共存

- 详情抽屉从最右侧滑出
- 抽屉宽度400px（与全局一致）
- 抽屉展开时，右侧面板被推到抽屉后面（或缩窄）
- 关闭抽屉后右侧面板恢复

## 十一、边界情况

### 11.1 区域内无任务无习惯

- 左侧面板显示空状态：「还没有任务，创建第一个吧」+ 创建按钮
- 习惯区域：「还没有习惯，创建一个吧」+ 创建按钮
- 角色显示空闲状态
- 数据面板全部显示0

### 11.2 空白自定义区域

- 使用通用简单场景（不随繁荣度变化）
- 无装饰商店（没有装饰可买）
- 其余功能（任务/习惯/数据面板）正常可用

### 11.3 面板展开状态的记忆

- 用户上次离开时面板是展开还是收起，下次进入同一区域时保持相同状态
- 每个区域独立记忆

### 11.4 窗口尺寸变化

- 面板宽度按百分比适应（左35%/中30%/右35%）
- 窗口极小时面板切换为纵向堆叠（左上/右下）
- 场景背景始终全屏铺满

### 13.1 任务模块补充字段
