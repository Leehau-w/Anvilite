## 三、已锁定的模块（6/10）

### 3.1 任务系统 ✓ (v1.1)

**文档**：`anvilite-task-module-spec.md`

两种事物类型：一次性任务（任务Tab）+ 周期性习惯（区域内部空间）

**一次性任务核心决策**：
- 必填：标题。选填带默认值：分类、难度(★★★)、优先级(中)、截止日期(明天,可无)
- 状态：待办 → 进行中(自动计时) → 已完成
- 难度5级+文字描述，优先级4级（紧急/高/中/低）
- 子任务最多6层，第一层缩进展示，更深层点击进入
- 创建：快速（顶部输入框回车）+ 完整（右侧抽屉）
- 分类选择器：按钮组（非下拉），预设区域直接显示，自定义在「其他▼」
- 完成：勾选框完成，4-5星需二次确认，卡片向右飞出
- 撤销：Toast(5秒)+已完成列表双入口，XP收回（允许降级），矿石不收回
- 删除：回收站30天清除。已完成可删（XP收回矿石不收回）或隐藏
- 列表：按状态分组，已完成不折叠带撤销。分类Tab筛选。可拖拽排序
- 过期任务淡红背景。分类删除须先移走任务
- **v1.1新增**：任务计时器（进行中自动计时）、矿石1:1获取

**周期性习惯核心决策**：
- 必填：名称、分类、难度、重复周期、预估时长
- 周期：每天/工作日/每周(可指定天+严格/灵活)/每两周/每月/自定义
- 连续性：完成+1、跳过-10%(最少-1)、错过减半(有容错则抵消)
- 容错：首次7连续得1个，之后14次得1个，最多持有1个，仅用于错过
- 可暂停(冻结连续)/归档/删除(用户选保留或收回XP)

### 3.2 设计基础 ✓ (v1.0)

**文档**：`anvilite-design-system-spec.md`

- **8套主题**：暗色4套(锻铁紫/矿石青/烈焰铸/翠林绿) + 亮色4套(晨光白★默认/薄荷纸/羊皮卷/云石灰)
- **优先级颜色**：紧急=#dc2626跨主题固定，高/中/低跟随主题
- **字体**：中文=Noto Sans SC，英文=Inter，数字=Inter SemiBold(600)+tabular-nums+letter-spacing 0.02em
- **标签**：12px，letter-spacing 0.03em，padding 5px 16px
- **字号**：H1=28/800, H2=20/700, H3=16/600, Body=14/400, Small=12/400, Tiny=10/400
- **间距**：4px网格，6级(xs=4, sm=8, md=12, lg=16, xl=24, 2xl=32)
- **圆角**：sm=4, md=8, lg=12, xl=16, full=9999
- **阴影**：3级(sm/md/lg)，亮色用阴影暗色用边框
- **按钮**：3类型(Primary/Secondary/Ghost) × 3尺寸(sm=28/md=36/lg=44)
- **输入框**：高度36px，聚焦2px accent边框+glow
- **分类选择器**：按钮组非下拉，一行放不下换行

### 3.3 XP与等级体系 ✓ (v1.1)

**文档**：`anvilite-xp-system-spec.md`

- **基础XP**：★=1, ★★=2, ★★★=3, ★★★★=5, ★★★★★=8
- **矿石(Ore)**：与XP 1:1同时获得，用于买装饰和解锁主题。撤销/删除时矿石不收回
- **加成**：按时+20%、连击+3%/天上限50%、高难(4-5★)+50%，可叠加
- **等级公式**：`round(5 × ln(level+1) × level)`，不封顶
- **节奏**：第一年每2周升一级，后续每月升一级
- **称号**：2套预设(锻造8级/经典RPG8级) + 用户自定义
  - 锻造：破壳/熔炼/锤炼/锋芒/极意/铸魂/锻石/不朽
  - 经典：新手/学徒/冒险者/游侠/英雄/大师/史诗/传奇

### 3.4 仪表盘 ✓ (v1.1)

**文档**：`anvilite-dashboard-spec.md`

- **布局**：卡片式自由布局（拖拽+调大小+可隐藏）
- **问候语**：时段变化（早上好/下午好/晚上好 + 角色名），不显示日期
- **5张卡片**：
  - 任务卡片（所有任务，简化：勾选+标题+优先级+截止日，进行中显示计时器）占左2/3
  - 习惯卡片（未完成的，非每日显示下次刷新时间）
  - 统计卡片（今日完成数/XP/连续天数）
  - 角色迷你卡（状态图标+文字+计时时长，等级+称号+经验条）
  - 快速创建卡片（独立，右侧）
  - 成长趋势卡片（统计摘要+热力图：月视图按天/周视图按4时段）
- **热力图颜色跟随主题accent色**

### 3.5 世界地图 ✓ (v1.0)

**文档**：`anvilite-worldmap-spec.md`

- **风格**：等距像素，羊皮纸背景，自由散布村落式
- **视角**：可滑动+可缩放(50%-200%)
- **区域总上限**：12个
  - 默认显示6个：家园(核心不可删) + 竞技场/书阁高塔/灵感工坊/锻造坊 + 里程碑殿堂(固定)
  - 隐藏模板6个：泉水/议事厅/远征大厅/观测站/植物园/广场
  - 用户可创建空白自定义区域（无繁荣度视觉）
- **区域标注**：名称 + 繁荣度★☆ + 微型进度条
- **交互**：单击→底部信息栏（名称/繁荣/任务数/XP/进度条/进行中任务/进入按钮）
- **繁荣度**：6级(荒芜/聚落/丰饶/繁荣/鼎盛/辉煌)，由技能维度等级驱动
- **12个区域独特的6级视觉变化**：全部定义完毕（辉煌级都融入锻石元素）
- **进入动画**：镜头推进zoom 800ms，cubic-bezier(0.4,0,0.2,1)
- **退出**：「← 返回地图」按钮(左上角) 或 ESC键，反向动画

### 3.6 区域内部空间 ✓ (v1.0)

**文档**：`anvilite-interior-spec.md`

- **布局**：像素场景全屏背景 + 半透明磨砂玻璃面板覆盖(rgba 75% + blur 8px + border)
- **面板默认隐藏**，按钮展开/收起（滑入/滑出300ms）
- **面板布局**：左侧~35%(任务+习惯) + 中间~30%(角色可见) + 右侧~35%(数据+收藏)
- **角色行为**：有进行中任务→工作动作(各区域独特)+场景动态效果；无任务→空闲动作+安静场景
- **角色状态只反映当前区域**的任务
- **任务计时器**：进行中状态自动计时，多任务可同时计时
- **左侧面板**：区域筛选的任务列表（简化版+计时器）+ 习惯列表（默认未完成，可切换全部+管理）
- **右侧面板**：技能等级+进度条、4统计(XP/完成数/投入时长/繁荣等级)、繁荣进度条、周对比、迷你热力图
- **收藏/装饰**：矿石余额显示、装饰商店按钮(模态框)
- **矿石系统**：与XP 1:1获得，买装饰+解锁主题皮肤，撤销/删除时不收回
- **装饰商店**：按繁荣等级分组解锁，矿石购买，每个区域独特装饰
- **主题解锁**：晨光白免费，其余7套用矿石解锁

## 五、关键决策日志

| 日期 | 决策 | 理由 |
|------|------|------|
| 03-27 | 项目命名 Anvilite · 锻石 | LifeForge已占用 |
| 03-27 | 从零重做第二版 | 第一版AI自由发挥质感不够 |
| 03-27 | 世界地图三层空间架构 | 养成感需要空间感 |
| 03-27 | 不做任何惩罚机制 | 核心设计红线 |
| 03-27 | 任务和习惯完全分离 | 两种本质不同的事物 |
| 03-27 | 基础XP定为1/2/3/5/8 | 小数字保持用户对每一点的感知 |
| 03-27 | 等级公式用对数曲线 | 保证前期快后期不爆炸增长 |
| 03-27 | 新增矿石(Ore)第二货币 | 避免用XP消费导致等级下降 |
| 03-27 | 矿石撤销/删除时不收回 | 努力的物质证明不应消失 |
| 03-27 | 8套主题（4暗+4亮），默认晨光白 | 用户自定义选择 |
| 03-27 | 优先级颜色：紧急固定红色，其余跟随主题 | 紧急需要最强警示 |
| 03-27 | 分类选择器用按钮组非下拉 | 比下拉更直观快捷 |
| 03-27 | 任务计时器（进行中自动计时） | 精确记录投入时长 |
| 03-27 | 区域重命名：书阁高塔/灵感工坊/锻造坊 | 更贴合RPG主题 |
| 03-27 | 12个区域模板（6默认+6隐藏）+ 空白自定义 | 开箱即用+高度可定制 |
| 03-27 | 辉煌等级全部融入锻石元素 | 品牌感统一 |
| 03-27 | 里程碑殿堂也有繁荣度变化 | 成就区域也应该有成长感 |
| 03-27 | 称号8级×2套+用户自定义 | 锻造主题+经典RPG双线 |
