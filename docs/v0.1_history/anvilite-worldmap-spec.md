# Anvilite · 锻石 — 世界地图模块规格文档

> **版本**: v1.0
> **日期**: 2026-03-27
> **状态**: 已锁定

---

## 一、模块定位

世界地图是Anvilite的核心养成体验。用户打开世界地图Tab看到的是一个属于自己的、持续生长变化的微型世界。每个区域对应一个技能/生活领域，随着任务完成而从荒芜走向辉煌。

---

## 二、地图基础设定

### 2.1 视觉风格

- **风格**：等距（Isometric）像素风
- **背景**：卷轴/羊皮纸底色（跟随主题变化，在羊皮卷主题下最贴合）
- **区域形态**：自由散布，区域像村落一样分散在地图各处
- **区域间连接**：装饰性的河流/树木/石头路连接区域，不可交互
- **氛围**：整体有手绘地图的质感，像冒险者的探索地图

### 2.2 视角控制

- **可滑动**：鼠标拖拽/触控板平移地图
- **可缩放**：鼠标滚轮/触控板缩放
- 缩放范围：50%~200%
- 默认缩放级别：让所有已激活区域都在视野内

### 2.3 区域位置

- **默认6个区域**：固定位置，系统预设排列
- **用户新增区域**：可拖拽放置在地图上任意位置
- 区域位置持久化保存

| 区域 | 对应分类 | 说明 |
|------|---------|------|

| 书阁高塔 | 学习/阅读 | — |
| 灵感工坊 | 兴趣/创造 | — |

| 区域 | 说明 |
|------|------|
| 里程碑殿堂 | 成就展示 + 角色属性入口，有繁荣度变化 |

**自定义模板区域（6个，默认隐藏，用户手动添加）：**

| 区域 | 适合场景 |
|------|---------|

### 3.2 默认地图状态（新用户）

新用户打开世界地图看到**6个区域**：
- 家园（中心偏上位置）
- 竞技场、书阁高塔、灵感工坊、锻造坊（围绕家园散布）
- 里程碑殿堂（较远位置，但可见）
- 所有区域初始为「荒芜」级别

### 3.3 区域管理

- **添加区域**：地图上有「+」按钮，点击后弹出选择面板，可从6个模板中选择或创建空白区域
- **删除区域**：必须先移走该区域下的所有任务和习惯（与任务系统规则一致）
- **改名**：所有区域可改名（里程碑殿堂除外）
- **移动位置**：用户新增的区域可拖拽，预设区域位置固定
- **区域总上限**：12个（含里程碑殿堂）

## 四、区域标注信息

### 4.1 地图上的常驻标注（每个区域上方/下方）

```
        [区域名称]
    [繁荣等级 ★★★☆☆☆]
[████████░░░░░░] 进度条
```

- **区域名称**：Small字号(12px)，fontWeight 600，text色
- **繁荣等级指示**：用★/☆表示6级，accent色（★）和border色（☆）
- **微型进度条**：到下一级繁荣的进度，宽度60px，高度3px，accent色填充

### 4.2 单击区域后的信息面板

**触发**：单击区域，区域高亮（发光边框），底部滑出信息栏

**信息栏内容**：

```
┌──────────────────────────────────────────────────────────┐
│ [区域名称]  繁荣: 丰饶 ★★★☆☆☆   [████████░░░] 到下一级  │
│                                                          │
│ 任务 5个 · 习惯 3个 · 累计 127 XP                         │
│                                                          │
│ 进行中: ▸ 完成日语N3语法笔记  ▸ 背单词30分钟               │
│                                                          │
│                                         [进入区域 →]      │
└──────────────────────────────────────────────────────────┘
```

- **位置**：屏幕底部，从下方滑入（动画250ms ease-out）
- **高度**：约120px
- **背景**：surface色，顶部边框1px border色
- **阴影**：lg（亮色主题）
- **区域名称**：H3字号(16px)，fontWeight 700
- **繁荣等级**：文字+★指示，accent色
- **进度条**：宽度150px，高度4px
- **任务数/习惯数/累计XP**：Small字号(12px)，数字用数字字体规则
- **进行中任务**：最多显示2个，Small字号(12px)，dim色，超出显示「...还有X个」
- **「进入区域」按钮**：Primary按钮 md尺寸
- **关闭**：点击地图空白处或再次点击同一区域关闭信息栏

### 4.3 区域悬停效果

- 鼠标悬停时区域轻微放大（scale: 1.05）
- 出现淡淡的发光边框（accent色 20%透明度）
- 过渡时间：150ms

## 五、繁荣度系统

### 5.1 驱动指标

繁荣度由**该区域对应的技能维度等级**驱动，与能力雷达图挂钩。

技能维度等级根据该区域累计XP计算（与角色等级使用相同的对数公式）：
```
技能等级所需XP = round(5 × ln(skillLevel + 1) × skillLevel)
```

### 5.2 繁荣等级与技能等级的映射

| 繁荣等级 | 中文名 | 技能等级范围 |
|---------|--------|------------|
| 1 | 荒芜 | Lv.0（未开始） |
| 2 | 聚落 | Lv.1-3 |
| 3 | 丰饶 | Lv.4-8 |
| 4 | 繁荣 | Lv.9-15 |
| 5 | 鼎盛 | Lv.16-25 |
| 6 | 辉煌 | Lv.26+ |

### 5.3 繁荣度升级时的反馈

- 地图上该区域播放升级动画（建筑变化过渡 + 发光特效）
- 底部Toast通知：「[区域名] 升级为 [繁荣等级名]！」
- 记录一条 GrowthEvent（type: area_level_up）

## 七、进入/退出区域动画

### 7.1 进入动画（镜头推进）

**触发**：点击信息面板的「进入区域」按钮

**动画流程**：
1. 信息面板收起（200ms）
2. 地图其他区域淡出（200ms）
3. 镜头向目标区域zoom推进（800ms，ease-in-out）
4. 区域建筑逐渐放大充满屏幕
5. 建筑淡出，内部空间淡入（最后200ms重叠）
6. 内部空间完全展示

**总时长**：约1000ms（信息面板收起和地图淡出与zoom并行）

**缓动函数**：cubic-bezier(0.4, 0, 0.2, 1)（Material Design标准ease-in-out）

### 7.2 退出动画（镜头拉远）

**触发**：点击左上角「返回地图」按钮

**动画流程**：进入动画的反向播放
1. 内部空间淡出，区域建筑淡入（200ms）
2. 镜头从区域zoom拉远回到地图视角（800ms）
3. 其他区域淡入（200ms）
4. 地图完全展示

**也可通过ESC键触发退出**

### 7.3 「返回地图」按钮

- 位置：区域内部空间的左上角
- 样式：Ghost按钮，sm尺寸
- 内容：「← 返回地图」
- 始终固定在左上角（不随内部空间内容滚动）

## 九、地图编辑模式

### 9.1 触发

地图右上角有「编辑」按钮（Ghost按钮），点击进入编辑模式

### 9.2 编辑模式下的功能

- 所有用户新增区域可拖拽移动位置
- 每个区域出现编辑按钮（改名/删除）
- 预设区域不可移动但可改名（里程碑殿堂除外）
- 背景变为网格辅助线
- 退出编辑模式：点击「完成编辑」按钮

## 十、边界情况

### 10.1 新用户首次打开

- 显示6个默认区域，全部荒芜状态
- 可选：首次打开时有简短的引导动画（镜头从远处慢慢zoom到地图，然后各区域依次亮起）

### 10.2 区域繁荣度升级

- 升级动画在地图视角播放（不需要进入区域内部）
- 建筑从旧形态渐变为新形态（500ms过渡）
- 升级完成后区域短暂发光（1000ms）
- 底部Toast通知

### 10.3 删除区域

- 必须先移走所有任务和习惯
- 删除后区域从地图消失（淡出动画300ms）
- 如果是模板区域：该模板重新回到「添加区域」的可选列表中

### 10.4 窗口尺寸变化

- 地图内容不缩放，通过滑动/缩放适应
- 缩放自动调整到让所有区域可见

## 十一、与其他模块的关系

| 交互 | 说明 |
|------|------|
| 世界地图 → 区域内部空间 | 点击「进入」zoom进入（下一个模块定义） |
| 世界地图 → 任务系统 | 区域=任务分类，新增/删除区域同步新增/删除分类 |
| 世界地图 → 能力雷达图 | 繁荣度由技能维度等级驱动，共享底层数据 |
| 世界地图 → 里程碑殿堂 | 里程碑殿堂是世界地图上的一个特殊区域 |
| 世界地图 → 仪表盘 | 角色迷你卡的状态参考当前进行中任务的分类 |
| 世界地图 → 成长事件 | 繁荣度升级记录GrowthEvent |
